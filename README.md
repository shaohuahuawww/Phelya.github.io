# 博客网站模板

这是一个纯静态的个人博客网站示例，当前首页布局参考了 `yccomic.site` 的信息层级重新设计，并保留本地二次元图片素材。

## 文件说明

- `index.html`: 页面结构
- `styles.css`: 响应式样式和视觉设计
- `assets/`: 本地角色图和舞台视觉素材

## 如何预览

直接用浏览器打开 `index.html` 即可。

如果你想用本地服务预览，也可以在当前目录运行：

```bash
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`。

## 当前首页结构

- 欢迎页覆盖层
- 顶部导航
- 搜索区
- 站点介绍
- 风向标导航
- 数学分区
- 站点动态
- 图库
- 学习计划
